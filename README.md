## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ docker compose up -d
$ yarn start:dev

# production mode
$ yarn start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
